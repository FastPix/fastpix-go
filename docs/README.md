# FastPix Go SDK Documentation

This directory contains comprehensive documentation for the FastPix Go SDK.

## Documentation Structure

### 📁 `analysis/`
Contains detailed analysis reports of the SDK and samples:
- `ANALYSIS_REPORT.md` - Comprehensive analysis of sample code quality and issues

### 📁 `summary/`
Contains summary documents and overviews:
- `FINAL_ANALYSIS_SUMMARY.md` - Final analysis summary of sample verification
- `EXAMPLES_SUMMARY.md` - Complete overview of sample implementations

### 📁 `models/`
Contains detailed documentation for API models and components:
- `apierrors/` - API error documentation
- `components/` - Component documentation  
- `operations/` - Operation documentation

### 📁 `sdks/`
Contains SDK-specific documentation for each service:
- `dimensions/` - Dimensions API documentation
- `drmconfigurations/` - DRM configurations documentation
- `errors/` - Error handling documentation
- `inputvideo/` - Input video documentation
- `invideoaifeatures/` - AI features documentation
- `liveplayback/` - Live playback documentation
- `managelivestream/` - Live stream management documentation
- `managevideos/` - Video management documentation
- `metrics/` - Analytics metrics documentation
- `playback/` - Playback documentation
- `playlist/` - Playlist documentation
- `signingkeys/` - Signing keys documentation
- `simulcaststream/` - Simulcast documentation
- `startlivestream/` - Start live stream documentation
- `views/` - Views documentation
