# SourceAccessMediaStatus

Determines the media's status, which can be one of the possible values.


## Values

| Name                                 | Value                                |
| ------------------------------------ | ------------------------------------ |
| `SourceAccessMediaStatusCreated`     | Created                              |
| `SourceAccessMediaStatusDownloading` | Downloading                          |
| `SourceAccessMediaStatusDownloaded`  | Downloaded                           |
| `SourceAccessMediaStatusValidating`  | Validating                           |
| `SourceAccessMediaStatusInQueue`     | In Queue                             |
| `SourceAccessMediaStatusProcessing`  | Processing                           |
| `SourceAccessMediaStatusReady`       | Ready                                |
| `SourceAccessMediaStatusFailed`      | Failed                               |
