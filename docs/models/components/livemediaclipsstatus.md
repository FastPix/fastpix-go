# LiveMediaClipsStatus

Determines the media's status, which can be one of the possible values.


## Values

| Name                              | Value                             |
| --------------------------------- | --------------------------------- |
| `LiveMediaClipsStatusCreated`     | Created                           |
| `LiveMediaClipsStatusDownloading` | Downloading                       |
| `LiveMediaClipsStatusDownloaded`  | Downloaded                        |
| `LiveMediaClipsStatusValidating`  | Validating                        |
| `LiveMediaClipsStatusInQueue`     | In Queue                          |
| `LiveMediaClipsStatusProcessing`  | Processing                        |
| `LiveMediaClipsStatusReady`       | Ready                             |
| `LiveMediaClipsStatusFailed`      | Failed                            |
