# UpdateMediaStatus

Determines the media's status, which can be one of the possible values.


## Values

| Name                           | Value                          |
| ------------------------------ | ------------------------------ |
| `UpdateMediaStatusCreated`     | Created                        |
| `UpdateMediaStatusDownloading` | Downloading                    |
| `UpdateMediaStatusDownloaded`  | Downloaded                     |
| `UpdateMediaStatusValidating`  | Validating                     |
| `UpdateMediaStatusInQueue`     | In Queue                       |
| `UpdateMediaStatusProcessing`  | Processing                     |
| `UpdateMediaStatusReady`       | Ready                          |
| `UpdateMediaStatusFailed`      | Failed                         |
