# DirectUploadVideoMediaMp4Support

Generates MP4 video up to 4K ("capped_4k"), m4a audio only ("audioOnly"), or both for offline viewing.



## Values

| Name                                                | Value                                               |
| --------------------------------------------------- | --------------------------------------------------- |
| `DirectUploadVideoMediaMp4SupportCapped4k`          | capped_4k                                           |
| `DirectUploadVideoMediaMp4SupportAudioOnly`         | audioOnly                                           |
| `DirectUploadVideoMediaMp4SupportAudioOnlyCapped4k` | audioOnly,capped_4k                                 |
