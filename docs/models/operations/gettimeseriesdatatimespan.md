# GetTimeseriesDataTimespan

This parameter specifies the time span between which the video views list should be retrieved by. You can provide either from and to unix epoch timestamps or time duration. The scope of duration is between 60 minutes to 30 days.



## Values

| Name                                       | Value                                      |
| ------------------------------------------ | ------------------------------------------ |
| `GetTimeseriesDataTimespanSixtyminutes`    | 60:minutes                                 |
| `GetTimeseriesDataTimespanSixhours`        | 6:hours                                    |
| `GetTimeseriesDataTimespanTwentyFourhours` | 24:hours                                   |
| `GetTimeseriesDataTimespanThreedays`       | 3:days                                     |
| `GetTimeseriesDataTimespanSevendays`       | 7:days                                     |
| `GetTimeseriesDataTimespanThirtydays`      | 30:days                                    |
