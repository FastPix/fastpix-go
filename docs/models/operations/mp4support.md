# Mp4Support

Generates MP4 video up to 4K ("capped_4k"), m4a audio only ("audioOnly"), or both for offline viewing.



## Values

| Name                          | Value                         |
| ----------------------------- | ----------------------------- |
| `Mp4SupportCapped4k`          | capped_4k                     |
| `Mp4SupportAudioOnly`         | audioOnly                     |
| `Mp4SupportAudioOnlyCapped4k` | audioOnly,capped_4k           |
